# PhoneTracker.ID

Website untuk melacak informasi nomor telepon Indonesia.

## Fitur
✅ Lacak operator dari nomor HP
✅ Deteksi lokasi berdasarkan prefix
✅ Tampilan responsive (mobile & desktop)
✅ Simpan hasil sebagai file
✅ Database operator terlengkap
✅ Tanpa API key (full client-side)

## Cara Deploy ke GitHub Pages

1. **Buat repository baru** di GitHub
   - Nama: `phonetracker-id` (atau sesuai keinginan)
   - Public repository

2. **Upload semua file** ke repository:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md` (opsional)

3. **Aktifkan GitHub Pages:**
   - Buka repository → Settings → Pages
   - Source: Deploy from a branch
   - Branch: `main` (root)
   - Save

4. **Akses website:**
   - URL: `https://username.github.io/repository-name`

## Penggunaan
1. Buka website
2. Masukkan nomor dengan format +62xxxxxxxxxxx
3. Klik "Lacak Nomor"
4. Lihat hasil instan

## Contoh Nomor
- +628123456789 (Telkomsel)
- +628561234567 (Indosat)
- +628781234567 (XL)
